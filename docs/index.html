Features

Everything you need, nothing you don't

🔒

Auto-Encryption

Automatically detects API keys, passwords, JWT tokens, and PEM keys. Encrypts them with ChaCha20-Poly1305 before they hit disk.

🔍

Fuzzy Search

Find any clipboard entry instantly. Type a few characters and ClipVault matches across your entire history.

⚡

Native Performance

Built in Rust with native macOS APIs. No Electron, no web views. Under 50 MB RAM and sub-100ms popup time.

🎨

Image Support

Clipboard images are automatically saved with TIFF-to-PNG conversion and thumbnail generation. Paste them back anytime.

⌨

Keyboard-First

Cmd+Shift+C opens your history. Navigate with arrow keys, search by typing, paste with Enter. Never leave the keyboard.

📅

Privacy by Default

7-day automatic cleanup. Your data stays on your machine. No accounts, no cloud sync, no telemetry.
